### **1. Core Principle**

Binary search exploits the **sorted nature** of the input data to eliminate half of the remaining elements at each step. Instead of checking every element (as in linear search), it compares the target value to the **middle element** of the current search interval:

- If equal → found.
- If target < middle → search the **left half**.
- If target > middle → search the **right half**.

This process repeats recursively or iteratively until the target is found or the search space becomes empty.
