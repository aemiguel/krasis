## 🔍 Core Idea

Given a **sorted** array `A` of length `n` and a target value `T`, binary search works by:
1. Comparing `T` to the **middle element** of the current search interval.
2. If equal → found.
3. If `T` is less → recurse (or iterate) in the **left half**.
4. If `T` is greater → recurse (or iterate) in the **right half**.

This halving continues until the target is found or the search interval becomes empty.
