# Quicksort Implementation in Rust

Here's a clean, idiomatic Rust implementation of Quicksort, including both in-place and functional variants, along with test cases and usage examples.

## 1. In-Place Quicksort (Mutable Reference)

```rust
fn quicksort_inplace(arr: &mut [i32]) {
    if arr.len() <= 1 {
        return;
    }

    let mut left = 0;
    let mut right = arr.len() - 1;
    let pivot = arr[left];

    while left < right {
        while left < right && arr[right] >= pivot {
            right -= 1;
        }
        while left < right && arr[left] <= pivot {
            left += 1;
        }
        if left < right {
            arr.swap(left, right);
        }
    }

    arr.swap(0, left);
    quicksort_inplace(&mut arr[..left]);
    quicksort_inplace(&mut arr[left + 1..]);
}
```
