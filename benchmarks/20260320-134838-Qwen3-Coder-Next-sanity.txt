## 🔍 Core Idea: Divide and Conquer

Binary search leverages the **sorted nature** of the input array to eliminate roughly half of the remaining elements in each step. Instead of checking every element (as in linear search), it compares the target value to the middle element:

- If equal → found!
- If target < middle → search the *left half*
- If target > middle → search the *right half*

This process repeats recursively (or iteratively) until the value is found or the search space is exhausted.
